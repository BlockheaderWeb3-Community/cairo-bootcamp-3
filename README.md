## Introduction to Building Starknet Dapps

# Setup

1. Clone `https://github.com/BlockheaderWeb3-Community/cairo-bootcamp-3/tree/dapp`

2. run `cd student-registry-dapp-starter`

3. run `npm i`

4. run `npm run dev`
